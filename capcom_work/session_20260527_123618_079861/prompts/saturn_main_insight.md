# コンテキスト (Context)
**マップタイプ**: 技術ランドスケープ (Saturn V - Telescope)
**分析対象**: 全体俯瞰マップ。
**手法**: SBERT (文章ベクトル化) + UMAP (次元圧縮) + HDBSCAN (クラスタリング)。
**視覚的エンコーディング**:

- **配置**: マップ全体の「形状」が技術空間の広がりを表します。
**目的**: マクロな視点で技術全体の構造を把握し、主要なテーマ（クラスタ）や未開拓領域（空白地帯）を発見すること。
